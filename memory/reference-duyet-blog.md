blog.duyet.net — Duyet runs it as a **digital garden** (evolving notes, not
polished essays), drafted in Obsidian.

Recurring themes (by volume):
- **ClickHouse** (largest): MergeTree/Replacing/Replicated engines, mutations,
  replication topologies, UDFs, Keeper, monitoring on Kubernetes, cheatsheets.
- **Rust**: async, building libraries (incl. for Next.js / WASM), language-model
  from scratch, ownership types (`Rc<T>`, `Saturating<T>`), dependency tooling.
- **AI agents / LLMs**: a leveled "build an agent" portfolio series — push-frontier
  AI, self-improving coding agent, personal life-OS agent, enterprise workflow
  agent — plus MCP, Ollama, workflow-vs-agents, `duyetbot`.
- **Data engineering**: Airflow (deferrable operators), DuckDB, OpenDAL.
- **Tooling & life**: homelab, Obsidian, personal analytics / life-logging.
